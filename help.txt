Top-level CLI entrypoint for `romm-cli`.

This binary can be used both as: - a **TUI launcher** (`romm-cli tui`), and - a **scripting-friendly CLI** for platforms/ROMs/API calls.

Usage: romm-cli.exe [OPTIONS] <COMMAND>

Commands:
  init       Create or update user config (~/.config/romm-cli/.env or %APPDATA%\\romm-cli\\.env) [aliases: setup]
  tui        Launch interactive TUI for exploring API endpoints
  api        Low-level access to any ROMM API endpoint
  platforms  Platform-related commands
  roms       ROM-related commands
  download   Download a ROM
  update     Check for and install updates for romm-cli
  help       Print this message or the help of the given subcommand(s)

Options:
  -v, --verbose
          Increase output verbosity

      --json
          Output JSON instead of human-readable text where supported

  -h, --help
          Print help (see a summary with '-h')

  -V, --version
          Print version
